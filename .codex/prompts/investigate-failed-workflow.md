# Investigate Failed Workflow

Use `.codex/workflows/ci-cd-failure-investigation.md`.

Required response:
1. Failing workflow/run/job.
2. Log evidence.
3. Root cause confidence.
4. Whether this is code, test, dependency, infra, permission, secret, or flaky timing.
5. Minimal fix proposal.
6. Verification plan.

Do not edit workflow YAML or source files unless the user approves.
