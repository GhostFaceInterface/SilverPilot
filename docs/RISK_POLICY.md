# Risk Policy

## Hard Safety Rules

- Real-money execution is forbidden.
- Bank automation is forbidden.
- Paper trades must go through the risk engine once implemented.
- Missing data should block automated paper-trade decisions.

## Initial Block Reasons

- Spread too high.
- Volatility too high.
- Daily loss limit reached.
- Weekly loss limit reached.
- Expected net gain does not exceed costs.
- Required data is missing.
